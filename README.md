# Cafe-stock
This script defines a menu list containing caf√© items, a stock dictionary containing stock values for each item, and a price dictionary containing prices for each item. Then, it calculates the total stock worth by iterating through the menu items, multiplying the stock value by the price for each item, and accumulating the total.
